Impressum

Die blaue Kladde ist ein von mehrere Autoren geschriebener privater Hobby-Koch-Blog aus Bochum.

Namensgebend für diesen Blog ist eine "blaue Kladde" in der über Jahre hinweg analog Rezepte gesammelt wurden von denen einige nun hier im Blog erscheinen.

Hinweise zum Datenschutz

Dieser Blog besteht im Wesentlichen aus statischen Webseiten die selbst keine Daten sammeln können. Er wird bei netlify betrieben. Netlify protokolliert u.a. Zugriffe / IP Adressen für einige Tage - siehe die Datenschutzerklärung von netlify für weitere Details.

Auf einigen Blog-Seiten wird zusätzlich ein dynamisches Kommentarsystem eingeblendet, welches auf Github Issues basiert. Das Kommentarsystem selbst sammelt keine Daten stellt jedoch eine Verbindung zu Github her. Für Github gelten entsprechend die Datenschutzhinweise von Github.
